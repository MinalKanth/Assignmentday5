# 📖 Overview

ColorShift JS is a lightweight web application that enables users to change the webpage background color dynamically using a dropdown menu.

The project introduces core front-end development concepts including:

* Event handling
* DOM manipulation
* User interaction
* Dynamic styling
* JavaScript functions

It serves as a practical example for beginners learning how JavaScript can directly modify webpage appearance.

# 🔍 How It Works

1. User selects a color from the dropdown.
2. The `onchange` event is triggered.
3. JavaScript captures the selected value.
4. The background color of the page is updated instantly.

Example:

```text
Select Color → Blue
```

Result:

```text
Background Color → Blue
```
